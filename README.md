The game consists of 8 interconnected enigmas (levels).

1.  **Solve the first seven Constants (C1-C7):** Each level presents a problem rooted in advanced Physics, Algebra, Chemistry, or Thermodynamics. The solution is always a numerical value.
2.  **Generate the Master Key:** The final (Eighth) constant requires concatenating the numerical solutions of C1 through C7, forming a 16-digit master key.
3.  **Complete the Protocol:** Use the 16-digit key to trigger the Master Reset and access the final coordinates (the gift).
